## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/MicrosoftEdge/Sonar-Website.git
$ npm install
```

Generate:

``` bash
$ npm run hexo-generate
```

Run server:

``` bash
$ npm start
```
