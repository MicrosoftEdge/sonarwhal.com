## Getting started

Clone the project:

``` bash
git clone https://github.com/sonarwhal/sonarwhal.com.git
```

Install dependencies:

```bash
npm install
```

Fetch remote content (e.g. documentation)

```bash
npm run update-content
```

Build the site:

```bash
npm run build
```

Run server:

```bash
npm start
```
